# CSS (Cascading Style Sheet)

- 3 ways
- Inline CSS (Hard to maintain)
- Internal CSS (Learning)
- Externals (real project)

# CSS Syntax

- selector{property:value}
- selector (it denotes html element, class, id, etc.)
- property
- value

# Color

     - color:red;
     - color:#000000; (hex code)
     - color: rgb(255, 0, 0) //red, green, blue
     - color: rgba(255, 0, 0,0.4)//red, green, blue, alpha
     - color: hsl(0, 25%, 20%)//hue, saturation, lighting
