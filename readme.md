# Git

## Configuration

- ssh-keygen
- add to github
- git clone url
- git config --global user.name "Your name"
- git config --global user.email "Your email"
- git config --list

## After Clone

- git branch
- git add filename
- git commit -m "add message here"
- git push origin main

## log

- git log
- git log --oneline

## Change branch Name

- git branch -M main (change master to main)

## Project Initialize

- create a folder (name demo) and open in vscode or open in terminal
- git init
- goto to github and make repository demo
- copy ssh url
- git remote add origin git@github.com:nareshtharu-dev/demo.git

## Step

- git status
- git add .
- git commit -m "your commit here"
- git branch
- git push origin branchName

# Branch

1. git checkout -b branchName(create and switch branch)
2. git checkout branchName
   or
3. git switch -c branchName(create an switch branch)
4. git switch branchName

## To check branch

1. git branch (for local)
2. git branch -r (remote branch)

## Branch Delete

1. To Delete local branch

- git branch -D branchName

2. To delete remote branch

- git push origin --delete branchName
